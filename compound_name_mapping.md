# COMPOUND NAME MAPPING — Abbreviations for Figures and Text

**Usage:** Reference for standardizing names in figures (publication) and text.
Long ECCC ESTS names → short abbreviations for axes, legends, tables.

---

## Abbreviation rules

1. **n-Alkanes:** keep as `n-C9`, `n-C10`, etc. (already short)
2. **Parent PAHs:** use ECCC abbreviation in parentheses: `Naph`, `Phe`, `Chr`, etc.
3. **Alkylated PAHs:** `C0-Naph` → `C₀Naph`, `C1-Phe` → `C₁Phe`, etc.
4. **Biomarkers:** use short code: `H30`, `H29`, `Ts`, `Tm`, `C₂₇αββ`, etc.
5. **Ratios:** keep slash format: `nC10/nC20`, `C₀Naph/C₄Naph`, etc.
6. **Subscripts:** use Unicode subscript characters (₀₁₂₃₄) in matplotlib figures
